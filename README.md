# CricBD IPTV Player

A modern, responsive, browser-based IPTV player for M3U/M3U8 playlists.

## Run locally

```bash
npm start
```

Then open `http://localhost:4173`.

The preview server has no external package dependencies. To use a different
port, run `PORT=8080 npm start`.

## Features

- M3U/M3U8 playlist parsing
- HLS.js and native HLS playback
- Search and category filters
- Persistent favorites and theme
- Custom playlist URL support
- Responsive Bengali interface
